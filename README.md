# IPL-Data-Analysis-Dashboard
IPL Analysis Dashboard (2008–2025) built using Power BI to provide season-wise insights into IPL history. Features include winners analysis, Orange Cap Purple Cap holders, interactive filters, player/team visuals, and dynamic dashboards using DAX, Power Query, and data modeling techniques.
- <a href="https://github.com/gowthamgowda69/IPL-Data-Analysis-Dashboard/blob/main/ipl_matches_data.csv">IPL Matches Dataset</a>
- <a href="https://github.com/gowthamgowda69/IPL-Data-Analysis-Dashboard/blob/main/players-data-updated.csv">Player Dataset</a>
- <a href="https://github.com/gowthamgowda69/IPL-Data-Analysis-Dashboard/blob/main/teams_data.csv">Teams Dataset</a>

### 📊 Project Insights 

🔹 Analyzed IPL data from 2008–2025 to uncover season-wise trends and player performances.
🔹 Identified season winners dynamically using interactive filters.
🔹 Highlighted Orange Cap and Purple Cap holders with team-wise insights.
🔹 Built KPI cards and visual dashboards for quick decision-making.
🔹 Used Power Query for data cleaning and transformation.
🔹 Applied DAX formulas for calculated measures and dynamic analysis.
🔹 Created an interactive and user-friendly dashboard experience using Power BI.

Tech Stack: Power BI | DAX | Power Query | Data Visualization | Data Analytics 🚀
### Conclusion

This IPL Analysis Dashboard (2008–2025) provides meaningful insights into team performances, season winners, and player achievements through interactive visualizations. By leveraging Power BI, DAX, and Power Query, the project transforms raw cricket data into actionable insights, making data analysis more efficient, engaging, and user-friendly. This dashboard demonstrates the power of data visualization in sports analytics and decision-making. 🏏📊🚀
